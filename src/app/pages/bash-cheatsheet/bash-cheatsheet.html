Bash Essentials

The actual daily toolkit for managing files, system permissions, and server environments.

1. Navigation

List files & folders in current directory: 
ls -la
Move into a specific folder: 
cd <folder-name>
Go up one folder level (exit directory): 
cd ../

2. Files & Editing

Create or edit a file directly in the terminal: 
nano <file-name>
Create a new folder: 
mkdir <folder-name>
Force delete a file or folder permanently: 
rm -rf <file-or-folder-name>

3. Admin & Documentation

Execute a command with root/admin privileges: 
sudo <command>
Open the official manual page for a command: 
man <command>
Display quick inline help summary: 
<command> --help
Filter outputs (crucial for hunting logs): 
<command> | grep <keyword>
